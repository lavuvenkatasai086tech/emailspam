How It Works

A classic NLP pipeline — from raw text to prediction in milliseconds

01
Raw Email Input
User pastes any email content — subject line, body, headers are all accepted.
02
Text Preprocessing
URLs, emails, phone numbers are tokenized. Stop-words removed, words lemmatized.
03
TF-IDF Vectorization
Text is converted to a 10,000-dimensional numeric vector using unigrams + bigrams.
04
ML Classification
Logistic Regression predicts spam probability with calibrated confidence scores.
05
Result + Insights
Verdict, confidence bars, detected spam signals, and risk level displayed in real time.
